HUGO 静态网站服务

Created At：2023-06-08 21:22 Updated At：2023-06-14 17:00
Keywords: hugo
  • 优点:GO语言,单文件,易发布
  • 即时性:当你发布网页之后,修改原始MD文件,网页会立刻随之变动

HUGO下载地址

https://github.com/gohugoio/hugo/releases
如果需要git来下载主题,还需要下载安装git,也可以不用git,直接下载
https://git-scm.com/download/win

运行

CMD CD 进入hugo目录

新建网站

hugo new site quickstart 此命令后会生成一个网站目录

添加path

每次path = %path%;d:\hugo\hugo或者进系统设置加path

新建文章,CD 进quickstart 目录

hugo new post\hugo.md

  • 此目录D:\HUGO\hugo\quickstart\content\post内会出现一个hugo.md文件
    默认发布的目录是网站目录的content\post内

主题下载

https://themes.gohugo.io/ 到官网寻找主题

  • 如果没有主题,则hugo无法显示任何网页,主题是必须的

1:使用git下载主题

  • CD 进quickstart 目录
    git clone https://github.com/alanorth/hugo-theme-bootstrap4-blog.git themes/hugo-theme-bootstrap4-blog
    git clone https://github.com/rujews/maupassant-hugo themes/maupassant

即git clone github主题网址.git themes/主题名

  • CD 进quickstart\themes目录
    git clone https://github.com/yihui/hugo-xmin.git

hugo-xmin 效果,极致精简


git clone https://github.com/avianto/hugo-kiera kiera
git clone https://github.com/serkodev/holy.git

2:或者直接在github下载zip压缩包后解压到quickstart\themes目录

主题使用

quickstart\hugo.toml
添加theme = "主题名"

编译网页,将md文件编译为html静态页面

hugo

启动网站

  • hugo server包含了编译和发布网站
    hugo server --theme=主题名或hugo server -t 主题名
    hugo server --theme=hyde
    如果已经在toml文件中指定主题,则可以直接hugo server启动网站

  • 注意:发布网页默认是不包含草稿的,而新建md时,默认其为草稿
    删去md文件中draft: true这句,表示其非草稿.这样网页才能显示出来
    hugo server --buildDrafts -t hyde 表示草稿也一起发布
    hugo server -t hugo-xmin
    hugo server -t hugo-theme-bootstrap4-blog

hugo server -t maupassant -p 1313
